reflex-activation path raises and is swallowed — **the test still

(9450) to the 2026-06-02 cycle**, so this is a pre-existing
test-double gap, not a regression from this gap's PRs.  Carry
forward: give the perf double a `_module_slots` attribute (or guard
the access) to silence the noise.
