# Space Survivalcraft

A space-themed survival crafting game built with Python and Arcade. Pilot a spaceship, gather resources, build a base, craft equipment, and fight enemies as you progress through technology-gated stages.

## Features

- Top-down 2D spaceship piloting on a scrolling 200x200 playing field
- Resource gathering from asteroids, gas fields, shipwrecks, and more
- Base building with stations, turrets, and force fields
- Ship customization with modular design
- Combat against enemy ships and space monsters
- Stage-based progression through technology unlocks

## Requirements

- Python 3.12
- Python Arcade 3.3.3

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/beknar/Space-Survivalcraft.git
   cd Space-Survivalcraft
   ```

2. Create and activate a virtual environment:

   ```bash
   python3.12 -m venv venv
   ```

   - **Windows CMD**: `venv\Scripts\activate.bat`
   - **PowerShell**: `venv\Scripts\activate.ps1`
   - **Linux/Mac**: `source venv/bin/activate`

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
